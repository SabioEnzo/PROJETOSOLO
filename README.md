O Projeto Solo é um site desenvolvido com foco em conteúdos relacionados à saúde, bem-estar, nutrição e qualidade de vida. O projeto apresenta páginas informativas com dicas de hábitos saudáveis, exercícios físicos e cuidados pessoais.

O objetivo principal é oferecer uma navegação simples, organizada e agradável para o usuário, utilizando apenas tecnologias front-end.
